# AI Skills Directory

This directory contains comprehensive documentation and learning materials for various AI skills and technologies implemented across the project.

## Structure

```
ai/
├── README.md              # This file
├── skills/                # Individual skill documentation
│   ├── pptx_generator/    # PPTX generation with PptxGenJS
│   │   ├── SKILL.md       # Learning objectives and roadmap
│   │   └── pptxgenjs.md   # Implementation details
│   ├── [skill-name]/
│   │   ├── SKILL.md
│   │   └── [implementation].md
│   └── ...
└── resources/             # General AI resources (future)
```

## Available Skills

### PPTX Generator with PptxGenJS
- **Location**: `ai/skills/pptx_generator/`
- **Description**: Learn to create PowerPoint presentations from YAML configurations using Node.js and PptxGenJS
- **Files**:
  - `SKILL.md`: Complete learning roadmap and objectives
  - `pptxgenjs.md`: Detailed implementation guidance and code examples

## Adding New Skills

To add a new skill to this directory:

1. Create a new directory under `ai/skills/` with a descriptive name
2. Add a `SKILL.md` file with learning objectives and roadmap
3. Add implementation-specific documentation files
4. Update this README with the new skill

## Skill Documentation Format

Each skill should include:

- **SKILL.md**: Learning objectives, prerequisites, implementation steps, and best practices
- **Implementation files**: Detailed technical documentation for specific technologies or approaches
- **Examples**: Code examples and usage patterns
- **Resources**: Links to external documentation and references

## Benefits

- **Centralized Learning**: All AI skills documented in one place
- **Structured Learning**: Clear progression from basics to advanced topics
- **Implementation Guidance**: Detailed technical documentation
- **Reusability**: Skills can be applied across different projects
- **Knowledge Sharing**: Easy to share and collaborate on skill development
